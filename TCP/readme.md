
# TCP
### script_TCPclient
### script_TCPserver
<br>
LAN2同士を接続した2台のMG400がTCP通信で
動作終了後に信号を送り合い、交互に作業を行う

<br>
<br>

## 実行方法
1. MG400をLAN2同士で接続する。
2. LAN1はそれぞれPCに接続し、サーバープログラム、クライアントプログラムの順に
   DobotStudio2020またはSCStudioでプログラムを実行する。

<br>
